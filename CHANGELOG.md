# 更新日志

## 2026-04-27

### 新增
- **stockpile 比对结果手动编辑库位** — 比对结果每条差异行旁增加"编辑"按钮，点击可修改本地库位并保存到数据库
- **一键覆盖全部库位** — 比对后有差异时，可一键将本地所有库位替换为导出文件的值
- **stockpile 搜索** — stockpile 管理面板增加搜索框，按条码/型号模糊搜索本地数据库
- **采购订单对接本地 stockpile.db** — 采购订单改为读取本地 SQLite 比对，无需每次手动上传 stockpile CSV
- **采购订单新条码一键入库** — 新条码确认后可一键写入本地 stockpile.db

### 修复
- stockpile 比对结果增加库位差异显示（此前只显示型号差异）
- 入库后下载 ZIP 时模板 CSV 丢失问题（savedNewEntries）

### 变更
- 采购订单上传流程简化：只需供应商 Excel 一个文件
